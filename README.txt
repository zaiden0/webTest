How to use:
1. Unzip the package.
2. Replace placeholder texts and SVGs in the images/ folder with your photos.
   - For best results, name photos photo1.jpg photo2.jpg photo3.jpg and update <img> src if needed.
3. Edit contact info (email, phone) in index.html.
4. To enable the contact form, replace the <form> action attribute with your form handler (Formspree, Netlify, server endpoint).
5. Upload the site folder to any static host (Netlify, Vercel, GitHub Pages, or your own hosting).
